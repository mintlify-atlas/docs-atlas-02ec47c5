Telegram Connector is a self-hosted REST API that wraps [Telethon](https://docs.telethon.dev/) with [FastAPI](https://fastapi.tiangolo.com/), so you can integrate Telegram features into any application without handling the MTProto protocol directly. Sign in with your Telegram account, list and join channels, and retrieve messages by time range — all through simple HTTP calls.

## How It Works

<Steps>
  <Step title="Obtain Telegram API credentials">
    Register your application at [my.telegram.org](https://my.telegram.org) to get an `api_id` and `api_hash`.
  </Step>
  <Step title="Run the service">
    Build and start the Docker container with your credentials. The service listens on port `5000`.
  </Step>
  <Step title="Sign in to your Telegram account">
    Call `POST /sign_in` with your phone number. Telegram sends a code to your Telegram app — submit it to complete authentication.
  </Step>
  <Step title="Start making API calls">
    With an active session, call `GET /channels` to list your channels or `GET /messages` to fetch messages from a time range.
  </Step>
</Steps>

<Note>
  Telegram Connector uses the MTProto user API (via Telethon), not the Bot API. This means you authenticate as a regular Telegram user account, giving you access to all channels and chats you're a member of.
</Note>
